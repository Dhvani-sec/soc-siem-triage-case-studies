# SOC SIEM Triage Case Studies

This repository contains documented alert investigations performed in a SIEM lab environment.

The goal is to strengthen structured triage methodology, analytical thinking, and documentation skills aligned with real-world SOC operations.

---

## Completed Case Studies

1. Double-Extension File Creation
2. Potential Data Exfiltration
3. Download from GitHub Repository
4. Spike of Domain Discovery Commands
5. Email Marked as Phishing after Delivery

---

Each case includes:
- Alert summary
- Investigation methodology
- Findings
- Final verdict
- Key lessons learned

---

Tools & Concepts Used:
- SIEM
- Log correlation
- Alert triage
- IOC validation
- Basic threat analysis
